The Literacy Service Plan is created annually by Project READ and the Literacy Service Planning Committee through Service Coordination and System Planning. Each Get SET (Skills, Education and Training) program for adults in Waterloo-Wellington is represented on the committee. We work in collaboration to develop a yearly Literacy Service Plan to meet the literacy needs of our community. For more information on the Local Get SET Programs that we work with, please see [The Directory of Programs](/programs-and-services/directory-of-programs/).
